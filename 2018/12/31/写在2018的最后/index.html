写在2018的最后


  2018 年，没有取得什么值得一提的成就，​与之前度过的一些同样空虚的年份相比，最大的不同可能在于由于开始热衷于电子资料的收藏而接触了比较大量的各类资源，尤其是视频资料，并借此以疏离现实世界。

  一些耳熟能详的传世经典无需再提，这里想谈谈自己所发现的一些趣味和惊喜，范围上基本都是今年的新作。

  关于电视剧，一开年就看到了 Unnatural 这样极佳的作品相当让人高兴，也因这部剧彻底成为了十元的粉丝。相比之下，虽然 Gakki 的作品也看过一些了，但一直还是路人心态 (请放心地抱走你们的 Gakki，不过十元必须是我的)。而在年末的时候，一部 My Brilliant Friend 闯入我的视线，只看一集就已让我沦陷，从而算是为 2018 年画上了圆满的句号。

  关于电影，那些大众化的大家容易接触到的就不太想说了。这里推荐一部我认为非常有趣和有意义的电影：大佛普拉斯。虽然是 2017 年的作品但我在看的时候确有一种相见恨晚的感觉。其构思非常巧妙，不论是镜头的运用还是叙事的方式都别开生面，一些场面对于观众和电影本身的联系做了深刻的思考，而黑白的质感配上恰当的旁白更是让人觉得意犹未尽。
  另外，虽然今年最棒的纪录片应当属于 BBC 的 Dynasties，但于我而言今年看的最有意义的纪录片是 Citizenfour，这是 2014 年的作品。当年棱镜计划被爆出来的时候，那个年纪的我们或许都没什么太大的感觉，而随着年龄的增长，有必要重新审视自己到底处在什么样的环境下，就我自己来说，我对如今的现实的情形感到非常失望和沮丧。
  如有兴趣，Unfriended: Dark Web 是一部题材新颖的恐怖片，对于我来说也算是一点惊喜。

  关于动漫，看的不算多，不过 宇宙よりも遠い場所 (比宇宙更遥远的地方) 是真的超赞，完成度很高，配乐完美，各方面的处理也很到位，三宅日向这个人物的性格完全击中了我，这是一种我很少体验到的感觉，(相比 SAO 中的 Asuna，我发现实际上我喜欢 Asuna 的发型，当然，还有长相，胜过了喜欢她的人设)，就 宇宙よりも遠い場所 的品质而言我大胆地设想今年应该没有比它更优秀的番了。

  差不多就这样了，一些对我来说真正意义极为重大足以改变人生轨迹的东西我是不愿意写出来的。我觉得向人宣扬这样的事物不管是对我还是对作品本身都是一种侮辱，相比之下，那些有缘的人自会与其相遇，不论是曾经还是未来。

  祝福各位，以及自己，都能在 2019 达成各自的向往和期待，至少平安健康，最好万事胜意。

2018.12.31
