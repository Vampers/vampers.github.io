你为什么不用 Telegram


  昨晚，在未事先通知的情况下停电了。什么事也做不了而又不想这么早睡觉的我，拿出手机，前往 Google Play，下载了 Telegram X，想看看这个同样官方出品的 APP 和我手机上那个不带 “X” 的原版 APP 有什么区别。

背景：Telegram 的客户端是开源的，理论上，任何人都可以打造出与众不同的 Telegram 客户端。而 Telegram X 最开始便是由第三方开发并维护，只是后来转为由官方进行维护。

  然后，我被攻陷了。毫无疑问这是我见过设计得最为出色的即时通讯应用，对比官方原版，Telegram X 不仅拥有同样强大无比的通讯功能，而且界面设计和动画显得更为优美。更重要的是，我在设置里看到这个应用居然还支持 Markdown ！天哪，这是怎样一种神奇的操作？（如果你还不知道 Markdown 是什么，那么我告诉你你正在看的这篇文章以及这个博客下的所有文章都是用 Markdown 写的，希望你可以直观地了解一下）

  在探索了一番 Telegram X 之后，再切回到熟悉的WX和扣扣，顿时产生了一种巨大的反差：“这简直就是一坨 💩”。如果可以，我希望以后再也不用这俩破玩意，让 Telegram X 成为我唯一的聊天应用。可事实上，我的手机通讯录里，有且仅有两个人在使用 Telegram，一位是我的学长，另一位是我的高中老师。

  一个显而易见的道理，对一个即时通讯软件来说，用户数量就是一切。而对于个人来说，这个效应更是连锁性的：“我的朋友都没用这个，所以我也用不着”……但即便如此，我还是希望，哪怕只有一个人，可以和我通过 Telegram 交流。

  这并不涉及任何所谓逼格或者国外的月亮比较圆之类的东西，只需要稍微了解一下 Telegram 的特点或者特色，很容易就能明白我为什么会有这种希望。

  随着年龄的增长，我已经数次体会到知音难觅是什么感受。而我之所以还抱有希望，只是因为我不愿意面对和承认自己这么多年来的失败。说失败其实也不太确切，因为这似乎的确挺难的。虽然像是自欺欺人，但是这有效就是了。在此之际，我很想把自己这些年来接触过的人做个描画。

   ▪ 第一种是虽有接触但并无太多交集的人。

   ▪ 另一种是在我周围的部分人，他们看起来很好，待人处事也很温和，性格也不错，但是很容易就能明白跟他们没办法有什么深入的交流。我不关心他们的想法，他们也是一样。如果跟他们稍微谈深一点，他们很可能把我当成异类或者觉得我是奇葩或者觉得我在故作深沉或是我在装逼或者是其他我也不知道的什么玩意，总之你终究是一个人在“表演”，他们并没有丝毫兴趣，只是看起来一切还是那么和谐。然而这并没有什么不妥，因为对每个人都很公平，谁也不欠谁的。

   ▪ 还有远远比我成熟的人，我的经历和他们比起来简直微不足道，在他们看来我还很幼稚，需要经历更多的历练。

   ▪ 当然还有那些我从心底里认可、能够完全信赖的人，有的平时可能也不怎么联系，各自有各自的生活，思考和研究的方向也不同，但是并不会有所疏离，只要跟他们说起自己的想法，他们就能试着去理解，这正是交流的基础。

   ▪ 另一位是我的前女友，当我们还在一起时，她竭尽所能的想要去理解我，想知道我在想什么在做什么，想了解我的经历、我的爱好和兴趣，这些都是基于她对我的爱，同样地，她也希望我也能这么对她。我和她当然可以毫无保留地谈及任何话题无论是深度还是广度，我们总能理解对方在说些什么考虑些什么，什么明的暗的有的没的我们都能明白。实际上，她是个相当完美的女朋友，甚至说妻子。只是，我们在彼此沉默时，都无法确定对方的心思。她很难过，他没察觉；他很脆弱，她不知道。

   ▪ 其实还有一种人，但是又不能跟前面几种很清楚地划分开来。我看不透他们的行为模式，他们思维独特，做事独立，有着不落凡尘的想法和见解，而他们的经历也往往让人惊叹。在他们面前，自己倒是有些相形见绌了。

  显然我不是一个善于社交的人，我似乎习惯于把自己包裹起来，这能给我安全感，但同时也让其他人很难走进我的内心。我不知道这是如何形成的，但在自我剖析的过程中我看到的好像就是这样，自己就像一只刺猬一般。而我认定了的人或事也往往很难改变，即使真的发生变故，对我来说那也会意味着一个极其痛苦的过程。

  在我看来，一个人的自我实现不在于有多少人认识他，也不在于他作出了多少贡献，而在于他去世时有多少人为他流下了真挚的眼泪。
